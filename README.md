# OpenGLScene
Scene for Computer Graphics class

## Controls

### Camera translation
    - X = A and D
    - Y = Q and Z
    - Z = W and S

### Camera rotation
    - X = UP and DOWN Arrows
    - Y = RIGHT and LEFT Arrows
    - Z = , and .

### Object translation
    - X = F and H
    - Y = R and V
    - Z = T and G

### Object rotation
    - X = I and K
    - Y = J and L
    - Z = U and O
